测试方法分两类：**购买前测试**和**购买后测试**。

### 购买前：用官方测试IP直接Ping

搬瓦工各机房均有官方提供的测试IP，你可以在本地终端直接ping，来判断延迟和丢包情况。

**主要机房测试IP：**

|------|--------|
| 洛杉矶 DC6 CN2 GIA-E（USCA\_6） | 173.247.96.1 |

| 香港 CN2 GIA（HKHK\_8） | 93.179.124.x（段内随机可测） |
| 日本大阪软银（JPOS\_1） | 173.254.223.1 |
| 荷兰联通9929（EUNL\_9） | 45.62.160.1 |

**Windows测试方法：**
打开命令提示符（CMD）或PowerShell，输入：

ping 173.247.96.1


**Mac/Linux测试方法：**
打开终端，直接输入：

ping 173.247.96.1


Ping结果里，time就是延迟（越低越好），Request timeout表示丢包（越少越好）。

想要更直观的多地测试，可以使用站长工具 ping.chinaz.com，输入测试IP，能看到全国各地到该机房的延迟分布图，一眼就能判断自己所在地区的连接质量。

### 购买前：下载测速文件

除了Ping测延迟，还可以尝试从官方测速服务器下载测试文件，来感受一下实际带宽。搬瓦工各机房均有对应的SpeedTest站点，可以测试下载速度、上传速度和延迟抖动。
